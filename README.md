# ResolveX – Web-Based Incident Resolution System

ResolveX is a modern web-based Incident Resolution System designed to efficiently manage, track, and resolve incidents within an organization.

It provides role-based access for users and administrators to report, monitor, and manage incidents in a streamlined and user-friendly interface.

---

## 🚀 Features

- User Authentication (Admin & User roles)
- Incident Reporting System
- Incident Status Tracking
- Admin Dashboard
- Responsive UI Design
- Clean and Modern Interface

---

## 🛠️ Technologies Used

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- Node.js
- npm
